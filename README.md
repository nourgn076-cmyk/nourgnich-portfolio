# Nour Gnich – Personal Portfolio Website

A modern, responsive personal portfolio website built with React, TypeScript, and Tailwind CSS.

## 🚀 Live Demo

[View Portfolio](https://nourgn076-cmyk.github.io/)

## 📋 Features

- **About Me** – Professional bio, profile photo, and social links
- **Experience** – Interactive timeline of professional roles
- **Featured Projects** – Clickable project cards with detail modals
- **Skills & Education** – Core competencies, technical skills, and academic background
- **Activities & Interests** – Achievements and personal interests
- **Research & Publications** – Master thesis on AI-Driven Business Intelligence
- **References** – Professional references with LinkedIn profiles
- **Contact** – Contact form and social links

## 🛠️ Tech Stack

- **React** + **TypeScript**
- **Vite** – Build tool
- **Tailwind CSS** – Utility-first styling
- **Shadcn/ui** – UI components
- **Lucide React** – Icons

## 📦 Installation & Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```
